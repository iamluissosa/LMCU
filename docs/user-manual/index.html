Bienvenida al ERP LMCU

Este manual ha sido diseñado para guiarte en el uso eficiente de LMCU, un sistema de gestión empresarial integral optimizado para la escalabilidad y el control operativo.

Soporte Técnico

Si encuentras algún problema no contemplado en esta guía, contacta al administrador de sistemas de tu empresa.

Interfaz de Usuario

La interfaz de LMCU está diseñada para ser intuitiva y productiva:

  1. Barra Lateral: Agrupa los módulos principales del negocio.
  2. Dashboard: Vista rápida de indicadores clave (KPIs).
  3. Modales de Acción: Ventanas emergentes para creación rápida de registros.

Configuración Inicial

Antes de comenzar a operar, es fundamental parametrizar los datos de la empresa para que los documentos fiscales se generen correctamente.

Configuración de Empresa (Settings)

Parámetro Descripción
RIF / NIT Identificación fiscal obligatoria para facturación.
Prefijos de Documentos Define cómo inician tus documentos (ej. COT-, FACT-, OC-).
Tasa de Cambio Valor del BCV o paralelo utilizado para la conversión multimoneda.
Importante

Cualquier cambio en la tasa de cambio afectará los nuevos documentos generados a partir de ese momento.

Usuarios y Roles

LMCU utiliza un sistema de control de acceso basado en roles (RBAC) con permisos granulares definidos en formato JSON.

Gestión de Usuarios

Pasos para crear un usuario:

  1. Dirígete a Configuración > Usuarios.
  2. Haz clic en "Nuevo Usuario".
  3. Completa el nombre, correo y asigna un Rol predefinido.
  4. Si es un vendedor, marca la casilla "Es Vendedor" para habilitar su tablero de comisiones.

Gestión de Clientes

El maestro de clientes almacena la información necesaria para el ciclo de ventas y la gestión de cobranzas.

Campos Clave

Campo Función
Tipo de Contribuyente Determina si el cliente es Agente de Retención de IVA.
Límite de Crédito Monto máximo en USD que el cliente puede adeudar.
Días de Crédito Plazo permitido para el pago de facturas.

Gestión de Proveedores

Crucial para el módulo de compras y el cálculo automático de retenciones de ISLR según la normativa local.

Retenciones de ISLR

Asegúrate de seleccionar correctamente el Tipo de Persona (PNR, PJND, etc.) para que el sistema aplique el % de retención según el Decreto 1808.

Productos y Servicios

LMCU distingue entre bienes físicos que afectan el stock y servicios profesionales.

Diferenciación de Ítems

  • Producto Físico: Requiere control de stock. Cada venta rebaja la existencia disponible.
  • Servicio (isService): No afecta el inventario. Ideal para mano de obra o consultoría.
Campo Descripción
Costo Promedio Calculado automáticamente con cada recepción de mercancía.
Precio Base Precio oficial de venta en USD.
Unidad de Medida Ej. Unidad, Caja, Metro, Hora.

Ciclo de Ventas

El flujo estándar de ventas en LMCU asegura la trazabilidad desde el primer contacto hasta el cobro.

Flujo Operativo

  1. Cotización (Quote): Documento informativo para el cliente.
  2. Orden de Venta (Sales Order): Reserva de compromiso. Valida stock si no es un servicio.
  3. Factura de Venta: Genera el compromiso legal y la cuenta por cobrar.
Validación de Stock

El sistema impedirá facturar productos físicos sin stock suficiente, a menos que el usuario tenga permisos de "Facturación en Negativo".

Ciclo de Compras

Controla el abastecimiento y las cuentas por pagar de la empresa.

Recepción de Mercancía (Goods Receipt)

Este es el paso que actualiza el inventario físico. Al recibir una Orden de Compra, el stock aumenta automáticamente.

Facturas de Compra

Registro de la obligación financiera. El sistema calculará automáticamente:

  • Retención de IVA (75% o 100% según proveedor).
  • Retención de ISLR (basada en el concepto de gasto).
  • IGTF (si el pago es en efectivo divisa).

Inventario y Almacén

Proporciona visibilidad en tiempo real de los activos de la empresa.

Kárdex de Producto

Consulta el historial detallado de cada ítem, viendo quién, cuándo y por qué se movió la mercancía.

Tesorería y Pagos

Gestión del flujo de caja (Cash Flow) de la empresa.

Métodos de Pago Soportados

  • Zelle / Transferencia Internacional.
  • Pago Móvil / Transferencia VES.
  • Efectivo (USD / VES) - Cálculo automático de IGTF.

Gestión Fiscal

Módulo especializado para el cumplimiento de las normativas del SENIAT.

Libros Fiscales

Generación automática en formato Excel/PDF de:

  • Libro de Compras.
  • Libro de Ventas.
  • Resumen de Retenciones de IVA e ISLR.

Reportes y Auditoría

Información estratégica para la toma de decisiones.

Log de Eventos
